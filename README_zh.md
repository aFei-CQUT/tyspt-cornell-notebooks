简单且实用的康纳尔笔记模板。

cornell-note是一个专为 Typst 设计的康纳尔笔记模板，旨在帮助用户记录高效且有组织的笔记。它提供了一个干净且直接的结构，使得记录和整理想法变得更加简单。
